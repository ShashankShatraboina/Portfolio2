## 🚀 Setup & Deployment

### GitHub Pages
1. Fork the repository  
2. Enable **GitHub Pages** in repository settings  
3. Your site will be available at `https://<your-username>.github.io/portfolio`

### Local Development
```bash
git clone https://github.com/aravinthvr/portfolio.git
cd portfolio
open index.html
````
